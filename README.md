# LKS 2015 - Module D

Pastikan port 80, 3306, 110, 25 dimatikan terlebih dahulu di komputer Anda.
Jalankan vagrant

```
vagrant up
```

Tunggu hingga proses selesai. Setelah vagrant berhasil di jalankan, buka browser dan akses http://localhost

```
email: sa@localhost
password: opensource
```
